# Traditional Rule-Based Chatbot

## Overview
This project implements a traditional rule-based chatbot using Python in Google Colab.

## Features
- Greeting users
- Displaying chatbot capabilities
- Assignment information
- Date and time responses
- Handling invalid input
- Exit functionality

## Technologies
- Python
- Google Colab

## How to Run
Open chatbot.ipynb in Google Colab and execute all cells.
